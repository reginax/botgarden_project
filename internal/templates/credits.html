This webapp is the result of a long collaboration between several institutions at UC Berkeley and beyond. Only a few of the contributors are listed here.

For questions about the content, and to access content beyond what is provided here, please contact Holly Forbes, hforbes@berkeley.edu.

This program was written and and is maintained by John Lowe (jblowe@berkeley.edu) at RIT/IS&T.

Comments, suggestions, bug reports, and so on are most welcome.

This program is written in Python using Django, jQuery, and accessing a Solr4 datasource. The data is extracted and updated nightly from the UCBG CollectionSpace system.
